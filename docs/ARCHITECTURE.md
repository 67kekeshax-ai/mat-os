# MAT OS — Архитектурное руководство

## Обзор

MAT OS — операционная система на базе ядра Debian, предоставляющая пользовательский
интерфейс в стиле Windows 10. Архитектура разделена на несколько независимых компонентов.

```
┌────────────────────────────────────────────────────────────────┐
│                        MAT OS Stack                            │
├────────────────────────────────────────────────────────────────┤
│  Приложения (внешние, скачиваемые отдельно)                   │
│  Calc / Paint / Notepad — не встроены в систему               │
├────────────────────────────────────────────────────────────────┤
│  MAT OS Shell (C++17 + X11/Xlib)                              │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────────────┐ │
│  │  Window  │ │ Taskbar  │ │  Start   │ │  File Manager    │ │
│  │ Manager  │ │  (48px)  │ │   Menu   │ │  (Explorer-like) │ │
│  └──────────┘ └──────────┘ └──────────┘ └──────────────────┘ │
├────────────────────────────────────────────────────────────────┤
│  MAT OS Login (C++ + /dev/fb0 framebuffer)                    │
│  Framebuffer login screen → запускает X + Shell               │
├────────────────────────────────────────────────────────────────┤
│  MAT OS Installer (C++ + /dev/fb0 framebuffer)                │
│  Лицензионная активация → разметка диска → debootstrap       │
├────────────────────────────────────────────────────────────────┤
│  Debian 12 Bookworm (ядро, libc, systemd, X.Org)             │
└────────────────────────────────────────────────────────────────┘
```

## Компоненты

### 1. Установщик (`installer/`)

**Технологии**: C++17, Linux framebuffer (`/dev/fb0`), POSIX

**Не зависит от X11** — работает на чистой консоли.

**Модули**:
- `gui/framebuffer.{h,cpp}` — прямой доступ к `/dev/fb0`, double buffering
- `gui/font8x16.inc` — встроенный растровый шрифт 8×16 (VGA ROM font)
- `gui/widget.{h,cpp}` — виджеты (Button, TextInput, ProgressBar, RadioGroup)
- `license/license.{h,cpp}` — проверка ключей (PRO=67691200, HOME=12006769)
- `disk/disk.{h,cpp}` — разметка диска через `parted`, `mkfs`, `mount`
- `install/install.{h,cpp}` — установка через `debootstrap`, настройка системы

**Шаги установки**:
1. Экран приветствия
2. Ввод лицензионного ключа (определяет редакцию)
3. Выбор диска назначения
4. Настройка пользователя (имя, hostname, пароль)
5. Разметка и форматирование диска
6. debootstrap базовой системы
7. Установка ядра и пакетов в chroot
8. Создание пользователя, настройка getty autologin
9. Установка GRUB
10. Готово → перезагрузка

### 2. Экран входа (`shell/src/login/`)

**Технологии**: C++17, Linux framebuffer, `libcrypt`, `shadow.h`

- Работает до запуска X (через `matos-login` — shell для `matos_login` user)
- Показывает часы, аватар пользователя, поле пароля
- Проверяет пароль через `/etc/shadow`
- После входа: `execvp("xinit", ...)` → запускает X + matos-shell

### 3. Оконный менеджер (`shell/src/wm/`)

**Технологии**: C++17, X11/Xlib

- `SubstructureRedirectMask` на root window = перехват всех MapRequest
- Каждое окно обёртывается в frame-окно с titlebar (32px)
- Кнопки: закрыть (E81123), развернуть (серый), свернуть (серый)
- Drag & drop за заголовок
- Resize за нижний/боковой край (4px border)
- Super+Left/Right = Snap (Aero Snap)
- Super+Up = Maximize

### 4. Панель задач (`shell/src/taskbar/`)

**Технологии**: C++17, X11/Xlib, double buffering (Pixmap)

- `override_redirect = True` — поверх всех окон
- EWMH `_NET_WM_STATE_ABOVE` для гарантии нахождения поверх
- Кнопка «Пуск»: 48px, только изображение (4-цветный логотип)
- Кнопки задач: 160px каждая, синий индикатор снизу для активного окна
- Часы: правый угол (ЧЧ:ММ / ДД.ММ.ГГГГ)
- «Show Desktop» — 16px крайний правый

### 5. Меню «Пуск» (`shell/src/startmenu/`)

**Технологии**: C++17, X11/Xlib

- Всплывает снизу над панелью задач
- Строка поиска с фильтрацией приложений
- Список установленных/доступных программ
- Кнопки питания: Завершить сеанс / Перезагрузка / Выключение
- Не содержит встроенных приложений — только запускалка

### 6. Файловый менеджер (`shell/src/filemanager/`)

**Технологии**: C++17, X11/Xlib, POSIX (`opendir`, `readdir`, `stat`)

- Боковая панель с закладками (~/Desktop, ~/Documents, ~/Downloads, /)
- Тулбар: Назад / Вперёд / Вверх / Адресная строка
- Список файлов: Имя | Размер | Дата изменения
- Иконки [D] для папок, [F] для файлов
- История навигации
- Клавиатура: ↑↓ = навигация, Enter = открыть, Delete = удалить, Backspace = вверх
- Открытие файлов через `xdg-open`

### 7. BSOD (`shell/src/bsod/`)

**Технологии**: C++, Linux framebuffer (без X11!), POSIX сигналы

- Устанавливает обработчики SIGSEGV, SIGABRT, SIGBUS, SIGFPE, SIGILL
- При любом сбое: переключается на tty1, рисует синий экран
- Синий фон `#0056A0`, белый текст
- Грустный смайлик :(, нарисованный через framebuffer примитивы
- Код ошибки + описание + timestamp
- Через 10 секунд → автоматическая перезагрузка

## Модульность

Каждый компонент (WM, Taskbar, StartMenu, FileManager, BSOD) можно
заменить без затрагивания ядра Debian. Единственный интерфейс — X11.

Новый WM можно написать и передать в `main.cpp` вместо `wm::WindowManager`.

## Ключи активации

| Редакция     | Ключ       | Файл              |
|--------------|------------|-------------------|
| MAT OS PRO   | `67691200` | /etc/matos-edition |
| MAT OS HOME  | `12006769` | /etc/matos-edition |

Ключ проверяется в `installer/src/license/license.cpp`.
Информация о редакции записывается в `/etc/matos-edition` после установки.
